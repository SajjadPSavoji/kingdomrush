# KingdomRush
# This project is a part of the course Advance Programming taught in University of Tehran.
# this is a simple game that categorized by attack and defence games
# there are two maps for game that can be chazen in the input file
# in the game there exists multiple places to place castles , each castle will attack the nearest enemy
# passing through routes.
#
#
# to run the project:
# > sudo chmode +x run.sh
# > ./run.sh
# > ./kingdom \rush < "inputfile.txt"
# where "inputfile.txt" contains details about map , types of enemies and the arrival time for each enemy.
# inputfile.txt can be test.txt or test2.txt
